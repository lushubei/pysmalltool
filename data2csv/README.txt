
使用说明:


一 目录结构

data2csv
---data    原始数据文件夹,把要处理的文件拷贝到该文件夹下即可,支持递归搜索,可多个文件夹及子文件夹

---out     处理结果输出文件夹,默认输出 文件为 s2p.csv , 常见excel软件可直接打开

---trans2csv.py  数据处理脚本



二 使用方法

1. 下载安装anaconda环境(windows环境,本质是需要其中(python3库 + pandas包)
https://repo.anaconda.com/archive/Anaconda3-2019.03-Windows-x86_64.exe

2. 命令行进入文件夹,data2csv

3. 运行脚本
python trans2csv.py
